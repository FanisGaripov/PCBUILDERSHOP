Name: "PC Configurator" website
This is a student project designed to teach basic html and python Flask skills.
The project is a website that has the functions of an online store, namely:
1. Registration
2. Authorization
3. The range of products divided into categories
4. The possibility of purchase (still conditional)
5. Profile with statistics on the website
6. Shopping Cart
7. Receipt of the receipt in the form of .json file
8. Adding products
9. Removal of goods.
This project requires the installation of the necessary libraries from requirements.txt
After installation, you need to download the archive, unzip and run the file server.py
